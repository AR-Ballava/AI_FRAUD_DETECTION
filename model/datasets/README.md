# Datasets

Place supervised training data in `training.csv` with columns:

```csv
text,label
"Urgent offer letter asks for refundable processing fee",fraudulent_offer_letter
```

Supported labels are:

- `fake_job_posting`
- `fraudulent_offer_letter`
- `scam_recruitment_email`
- `suspicious_terms_conditions`
- `company_fraud_signals`
